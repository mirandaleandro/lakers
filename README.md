Lakers + Clippers installation

Getting Started
-------------

### Clone the repositories

```
git clone https://github.com/mirandaleandro/lakers.git
git clone https://github.com/mirandaleandro/clippers.git

```

### Install dependencies

> ``` 
> cd lakers
> npm install
> bower install
> ```

### Build and Run

> ``` 
> cd lakers
> npm install
> bower install
> ```
