#Lakers
This project back-end can be found at [Clippers](https://github.com/mirandaleandro/clippers.git)

Getting Started
-------------

### Clone the repositories

```
git clone https://github.com/mirandaleandro/lakers.git

```

### Install dependencies

> ``` 
> cd lakers
> npm install
> bower install
> ```

### Build and Run

> ``` 
> gulp serve
> ```
